You need to install node.js

in DOS go to the directory containing package.json and type npm start

if it doesn't come up automatically, then type in the browser URL box http://localhost:[port nbr] where port nbr is specified in the package.json file (usually it's arround 3000)

npm

if you get a ENOTFOUND cannot connect to http://localhost:8080, then try npm config delete proxy

ensure your have .npmrc and .typeingsrc files inside your Angular project root folder before running npm [args]

.npmrc

proxy=[your proxy server address, if you have one]
https-proxy=[same as proxy]
strict-ssl=false
registry=http://registry.npmjs.org

.typingsrc

proxy=[your proxy server address, if you have one]
rejectUnauthorized=false
